# Introduction to DBMS

## What is DBMS?
A Database Management System (DBMS) is software that allows users to
define, create, store, retrieve, and manage data in a database.

## File System vs DBMS

| File System | DBMS |
|------------|------|
| Data redundancy | Reduced redundancy |
| No security | High security |
| No backup | Backup & recovery |
| Data inconsistency | Data consistency |

## Advantages of DBMS
- Reduced data redundancy
- Data security
- Data consistency
- Backup and recovery
- Easy data access
